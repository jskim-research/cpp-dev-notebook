# C++ Study & Practice

이 레포지토리는 C++ 문법 정리, 객체지향 프로그래밍 개념 학습, 그리고 코딩테스트 대비 문제 풀이를 기록하기 위한 개인 학습 저장소입니다.

## 📌 목적
- C++ 문법 및 핵심 개념 정리
- 객체지향 프로그래밍(OOP) 예제 작성 및 연습
- 알고리즘 문제 풀이 (백준, 프로그래머스 등)
- 코딩 인터뷰 및 기술 면접 대비
